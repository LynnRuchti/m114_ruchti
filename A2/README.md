*   **Media-Codecs**  
    CoDec → **Co**dierung**Dec**odierung → MP3-Code, MPEG2-Code, MPEG-4-AVC-Code, Reed-Solomon-Code (Audio-CD)
*   **Geheimcode**  
    Mit monoalphabetischer Verschlüsselung werden Buchstaben durch  
    andere Buchstaben ersetzt (Verschlüsselungsverfahren)
*   **ISBN-Code**  
    International Standard Book Number zur Identifizierung von Büchern
*   **Genetischer Code**  
    Eine biologische Anleitung, nach der während der Proteinbiosynthese die  
    Informationen der DNA/RNA in Aminosäuresequenzen übersetzt werden

**! Codes =/= Datenformate/Protokolle**

CODE (=entzifferbar) = Wenn man Zeichen durch andere Zeichen ersetzt (kann systematisch sein), z.B. T -> 2, E -> r, S -> + : TEST <=> 2r+2<br>
kann benutzt werden um Übertragungsgeschwindigkeit, Speicherplatz zu fördern oder eine Sicherung gegen Verfälschungen (Übertragungs-/Verarbeitungsfehler einschränken/verhindern) & unerlaubten Zugriff<br>


Gleitkommazahlen IEEE 754 = Zahlen, die binär gespeichert, aber beim Speichern/Auslesen (durch Recheneinheit der CPU) **codiert/decodiert** werden müssen<br>
ermöglicht Zahlenformate mit grossem Wertebereich & gleichmässiger relativer Genauigkeit über den Wertebereich, besteht aus fester Anzahl Ziffern (Single~7, Double~15)<br>
